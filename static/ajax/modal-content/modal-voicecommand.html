Comando de voz

Principais comandos
  • Você pode, ao invés de clicar no menu, apenas chamá-lo, em um tom de voz lento e limpo, usando a melhor dicção possivel. Exemplos: produtos, produtos ativos, colaboradores, clientes pessoa física.
  • Desligar som - Deixa de emitir som para as notificações.
  • Ligar som - Emite som para as notificações.
  • Desligar microfone - Desativa o microfone.
  • Ajuda - Exibe este modal com a listagem de comandos.
  • Fechar modal - Fecha este modal com a listagem de comandos.
  • Fechar menu - Fechar menu(se o menu não estiver visível, o mesmo será mostrado).
  • Mostrar menu - Mostrar menu(se o menu já estiver visível, o mesmo será fechado).
  • Para cima - Navega direto para o topo da página.
  • Para baixo - Navega direto para o fim da página.
  • Voltar - Volta para a pagina anterior.
  • Sair - Desloga o usuário do sistema.
